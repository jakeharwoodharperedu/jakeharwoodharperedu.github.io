Completing TutorialsPoint
In the Internet Fundamentals/Networking lesson, my first activity was completing the
tutorials from TutorialsPoint.
These tutorials covered Internet Reference Models, Internet Protocols,
and the Domain Name System (DNS). The Internet Reference Models tutorial provided a detailed explanation
of the OSI model, explaining the seven layers and their functions.
The Internet Protocols tutorial focused on the various protocols that facilitate communication
over the internet, such as TCP/IP, UDP, and HTTP.
The DNS tutorial explained how domain names are translated into IP addresses,
making it possible for users to access websites using human-readable names.

Verifying Internet Connectivity
The second activity I did involved verifying internet connectivity.
This involved using ipconfig on the command prompt, and using a trace route website.
I used the ping command to test local and remote connectivity by pinging my local IP address,
the default gateway, and an internet host 8.8.8.8.
These activities provided practical skills in diagnosing and confirming network connectivity issues.
