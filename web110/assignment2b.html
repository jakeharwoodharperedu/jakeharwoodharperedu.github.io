Activity 2: Practice with Google Search Operators

Google Search Operators seem to be an incredibly useful and important tool.
Through practicing with these operators, I learned how to use specific commands like "site:", "filetype:", "intitle:",
and "inurl:" to narrow down search results. For example, using "site:wikipedia.org" will make it so the Google
search only displays web pages within Wikipedia, and "filetype:pdf" filters the search results to only show
PDF document files, which can signifcantly
reduce the time spent sifting through irrelevant results.

Application:
Youtube recently drastically changed their search feature to be much worse, so I could use
Google searches with the use of search operators like "site:youtube.com" to effectively do a
Youtube search, but on Google.
In academic research usage of these search operators can help locate specific information like
scholarly articles or case studies quickly.
Knowing how to effectively use these search operators can improve my overall ability and efficiency
in navigating the vast amount there is online.
